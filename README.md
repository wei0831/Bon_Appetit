# Bon Appetit - Restaurant Framework
CodingDojo 2015 Full Mean Stack Project

Modified Date: 07/27/2015

##To Run
Run the following command
```
npm install && bower install & grunt default
```

##Technology Used
- Nodejs
- Express.js
- AngularJs
- MongoDB
- Bootstrap
- JWT (json web token)
